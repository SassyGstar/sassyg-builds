# GIA Academy — Hub

The GIA-branded landing page for **GIA Academy** (Governed. Intelligent. Accountable.), part of GIA / ACE Advising. This is a separate section from the OGO Financial Services builds in this repository.

## What's here

A single-file hub page (`index.html`) that gathers GIA Academy offerings in one place:

- **AI Readiness Assessment** — links to [`../gia-academy-ai-readiness/`](../gia-academy-ai-readiness/)
- **The GIA Course** — links to gia.aceadvising.com
- **Enterprise Control Assessment** — links to gia.aceadvising.com

It's designed to grow: new GIA content can be added as additional cards without touching the OGO homepage.
