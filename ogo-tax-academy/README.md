# OGO Tax Academy

A self-contained training web app for tax preparers getting AFSP-ready
(Annual Filing Season Program), including:

- 📚 Lesson tracks (Foundations, Income Recognition, Credits & Deductions)
- 🧪 Practice tests with scoring
- 🧾 Schedule C Decision Lab
- 📜 AFTR comprehension practice
- ⚖️ Circular 230 & preparer due diligence
- 🎓 Completion certificates

## Try it

Open `index.html` in any web browser. An internet connection is required the
first time (the page loads its display libraries from a CDN).
